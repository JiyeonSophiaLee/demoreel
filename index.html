Table scene

This scene is the first shot for my solo project.

The big interesting things were PBR shader, SSS shader and render settings.

I didn't model 80% objects in the scene, but I retextured 95% of them to create the color and texture for maching the mood.

Chandelier scene.

Rendering refracted objects took long time.

For saving render time, in 2 shots out of 3 shots in this scence, I just rendered only each frame for each shots.

I brought each frames into nuke, and installed a camera to make them animated. There were some flaws, so I compensated with defects in Nuke.

SunRise scene

This shot shows whole environment of the project.

There isn't only xgen but lots of other materials, such as fur, wood, fabric, sss objets. It was first time using Substance Designer.

I used fur library first to create randamly guid curves on the rug, then brought them into zbrush to comb them.

Finally, I wrap up with curve with xgen in Maya.
